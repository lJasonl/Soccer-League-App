# Soccer League App

Mobile-first youth soccer league management platform.

## Technology Stack

### Frontend
- Flutter
- Riverpod
- GoRouter
- Material 3

### Backend
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic

### Storage
- Cloudflare R2

## Features

### Core
- Seasons
- Divisions
- Teams
- Rosters
- Games
- Standings
- Playoffs

### Administration
- League Configuration
- Role Management
- Scheduling
- Field Management
- Registration Settings
- Payment Settings

### Mobile
- Parent Portal
- Coach Portal
- Referee Portal
- Notifications
- Team Messaging

### Future Enhancements
- Tournament Management
- Background Checks
- Volunteer Tracking
- Sponsorship Management
- Advanced Analytics
